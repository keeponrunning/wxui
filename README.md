# wxui

weex ui components.

# Preview

![Demo](https://github.com/keeponrunning/wxui/blob/master/assets/wxui.gif)

# Run

1. `$ git clone`
2. `$ cd wxui`
3. `$ npm install`
4. `./start`
5. **down [playground](https://weex.apache.org/cn/playground.html)**
6. open http://yourIP:4040
7. scan
